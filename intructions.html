Instrucciones Paso a Paso

Paso 1: Ir a la Tienda

Navega a la tienda de Quiz Cristiano para ver los productos disponibles.

Paso 2: Seleccionar un Producto

Elige cualquier avatar premium o pack de avatares. Los precios están en dólares pero se convertirán a pesos argentinos automáticamente.

Paso 3: Iniciar el Proceso de Pago

  1. Haz clic en "Comprar" en el producto deseado
  2. Se abrirá un modal con opciones de pago
  3. Selecciona "Mercado Pago"
  4. Serás redirigido a la página de pago de Mercado Pago

Paso 4: Completar el Pago

  1. Usa una de las tarjetas de prueba listadas arriba
  2. Completa los datos del titular según las instrucciones
  3. Confirma el pago
  4. Serás redirigido de vuelta a Quiz Cristiano

Paso 5: Verificar el Resultado

Después del pago exitoso:

  • Deberías ver una notificación de éxito
  • El producto se desbloqueará automáticamente
  • Podrás usar el avatar premium en tu perfil

Qué Esperar Durante las Pruebas

✅ Pago Exitoso

  • Redirección a Quiz Cristiano con mensaje de éxito
  • Avatar desbloqueado inmediatamente
  • Disponible en el selector de perfil
  • Webhook procesado correctamente

❌ Pago Rechazado

  • Mensaje de error en Mercado Pago
  • Redirección de vuelta con estado de fallo
  • Producto permanece bloqueado
  • Posibilidad de intentar nuevamente

⏳ Pago Pendiente

  • Estado de "procesando" en Mercado Pago
  • Webhook recibirá actualización posterior
  • Producto se desbloqueará cuando se apruebe

Solución de Problemas

La página de Mercado Pago no carga
Verifica que tengas conexión a internet estable. Si el problema persiste, prueba en una ventana de incógnito o limpia el cache del navegador.
El pago fue aprobado pero el producto no se desbloqueó
Espera unos minutos para el procesamiento del webhook. Si continúa el problema, verifica la consola del desarrollador para errores.
Error al redireccionar de vuelta a Quiz Cristiano
Esto puede indicar un problema con las URLs de retorno. Verifica que estés en el dominio correcto y que las URLs estén bien configuradas.

Información Técnica

URLs de Prueba

  • Sandbox: Habilitado para todas las transacciones
  • Webhook: Configurado para recibir notificaciones
  • URLs de retorno: Dinámicas según el dominio actual
