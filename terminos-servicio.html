Términos de Servicio

Condiciones de uso de Quiz Cristiano

Última actualización: 19 de junio de 2025

¡Bienvenido a Quiz Cristiano! Estos términos establecen las reglas para usar nuestra aplicación. Al usar Quiz Cristiano, aceptas estos términos completamente.

1. Aceptación de Términos

Al acceder y usar Quiz Cristiano, confirmas que:

  • Has leído y entendido estos términos
  • Aceptas cumplir con todas las condiciones
  • Tienes la edad legal para aceptar contratos en tu jurisdicción
  • Si eres menor de edad, tienes permiso parental

2. Descripción del Servicio

2.1 ¿Qué es Quiz Cristiano?

Quiz Cristiano es una aplicación web educativa que ofrece:

  • Juegos de preguntas bíblicas: Preguntas del Antiguo y Nuevo Testamento
  • Sistema de progreso: Monedas virtuales, logros y ranking
  • Contenido educativo: Material de aprendizaje cristiano
  • Experiencia personalizada: Perfiles y configuraciones

2.2 Acceso al Servicio

Ofrecemos acceso a través de:

  • Navegadores web modernos
  • Aplicación web progresiva (PWA)
  • Dispositivos móviles y computadoras

3. Registro y Cuentas de Usuario

3.1 Creación de Cuenta

Puedes usar Quiz Cristiano de dos formas:

  • Usuario invitado: Sin registro, datos locales
  • Usuario registrado: Con cuenta Google, sincronización en la nube

3.2 Responsabilidades del Usuario

Si creas una cuenta, eres responsable de:

  • Mantener la seguridad de tu cuenta
  • Proporcionar información precisa
  • Notificar cualquier uso no autorizado
  • Cumplir con estos términos

3.3 Restricciones de Cuenta

Está prohibido:

  • Crear múltiples cuentas para obtener ventajas
  • Compartir credenciales de cuenta
  • Usar bots o automatización
  • Intentar acceder a cuentas de otros usuarios

4. Uso Aceptable

4.1 Comportamiento Permitido

Quiz Cristiano está diseñado para:

  • Educación y entretenimiento familiar
  • Aprendizaje sobre la Biblia y fe cristiana
  • Uso personal y no comercial
  • Compartir conocimientos bíblicos

4.2 Comportamiento Prohibido

Está estrictamente prohibido:

  • Hacer trampa: Usar métodos deshonestos para obtener puntuaciones
  • Explotación: Abusar de bugs o vulnerabilidades
  • Contenido inapropiado: Crear nombres de usuario ofensivos
  • Interferencia: Intentar interrumpir el servicio
  • Ingeniería inversa: Descompilar o modificar la aplicación
  • Uso comercial: Revender o redistribuir contenido

5.1 Nuestro Contenido

Quiz Cristiano y todo su contenido son propiedad de los desarrolladores, incluyendo:

  • Preguntas y respuestas

  • Código fuente

  • Mascota "Joy" y personajes

5.2 Contenido Bíblico

Las referencias bíblicas utilizadas provienen de:

  • Dominio público: Textos bíblicos tradicionales
  • Interpretaciones comúnmente aceptadas
  • Conocimiento teológico general

5.3 Tu Contenido

Si proporcionas contenido (como nombres de usuario), garantizas que:

  • Es apropiado y no ofensivo
  • No infringe derechos de terceros

El manejo de tus datos personales se rige por nuestra Política de Privacidad, que forma parte integral de estos términos.

6.1 Datos del Juego

  • Prevenir trampas

7.1 Monedas del Juego

  • No son reembolsables: Una vez gastadas, no se devuelven
  • Pueden perderse: Si eliminas tu cuenta o datos

Actualmente, Quiz Cristiano es completamente gratuito. Todas las funciones están disponibles sin costo. Si en el futuro implementamos compras opcionales, se actualizarán estos términos.
